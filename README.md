# Building with react

## How to use

1. clone專案後，開啟cmd路徑移置該專案目錄，下載node_module</br>
```
npm install
``` 
<img src="Screenshot/1.png" width="650">

2. 之後打開瀏覽器輸入localhost:3000就可以同步開發囉，當您修改程式碼會立即在網頁上做更改不必另外鍵入npm run build</br>
```
npm start
```
<img src="Screenshot/2.png" width="650">
